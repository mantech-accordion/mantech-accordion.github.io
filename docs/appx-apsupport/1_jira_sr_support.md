## 개요

본 가이드는 Accordion 플랫폼을 사용하며 발생하는 다양한 기술적 요구사항을 신속하고 정확하게 해결하기 위한 SR시스템 이용 방법을 안내합니다.  
SR은 Accordion과 관련된 모든 기술 문의, 작업 요청, 장애 접수를 처리하는 공식 기술 지원 창구입니다.  
  
Accordion 기술 지원 조직은 고객의 요청에 단 한 순간의 공백도 없이 대응하기 위해 **전체 알림 시스템**을 운영하고 있습니다.

접수된 모든 SR 건은 즉시 엔지니어링 조직 전체에 공유됩니다.  
이는 특정 담당자의 부재나 작업 지연이 발생하더라도, 대기 중인 다른 전문 엔지니어가 즉각적으로 이슈를 파악하고,  
신속하게 답변과 해결책을 제공할 수 있도록 하기 위함입니다. 고객의 소중한 시간을 아끼고, 중단 없는 서비스를 지원합니다.

## SR 티켓팅 워크플로우

![sr_workflow](/assets/images/appx/support/sr_workflow.png){: width="1200" }

**SR 기술 문의 같은 경우 아래와 같은 상태를 가집니다.**
1. 지원 대기 중 : 문의가 접수된 직후의 상태
2. 진행 중 : 문의사항을 확인하였으며, 답변이 작성된 상태
3. 보류 : 내부적으로 확인이 필요할 경우 답변이 작성되기 전 임시 상태
4. 취소 : 문의가 취소된 상태
5. 해결됨 : 문의한 사항이 해결된 상태

## SR 계정 요청 방법

![sr_main](/assets/images/appx/support/sr_mail.png){: width="700" }

담당자 이메일 또는 고객사의 대표 이메일을 맨텍솔루션 담당 엔지니어에게 전달해 주시면, 해당 주소로 SR 시스템 초대 메일을 발송해 드립니다.  
수신하신 메일 본문의 링크를 통해 계정 생성을 완료하신 후 시스템을 이용하실 수 있습니다.

## SR 메인 화면

![sr_main](/assets/images/appx/support/sr_main.png){: width="1200" }

- Accordion 기능 문의
  - Accordion의 주요 기능, 상세 사용법 및 설정 방법 등 플랫폼 활용에 대한 궁금증을 해결해 드립니다.

- Accordion 설치/제거 문의
  - 신규 클러스터 설치, 기존 환경 제거 및 버전 업그레이드와 관련된 기술 지원을 요청하실 수 있습니다.

- Accordion 장애 문의
  - 서비스 사용 중 발생한 오류, 성능 저하 등 비정상적인 이슈에 대해 신속한 조치를 지원합니다.

- Accordion 유지보수 문의
  - 안정적인 서비스 운영을 위한 정기 점검, 패치 적용 및 리소스 추가 증설 등을 문의하실 수 있습니다.

## Accordion 기능 문의

![sr_function](/assets/images/appx/support/sr_function.png){: width="1200" }

제목: 문의의 핵심 내용을 직관적으로 파악할 수 있도록 요약하여 기재해 주세요.

공유 대상: 진행 상황을 공유받을 소속 그룹(고객사 단위)을 지정합니다. 지정된 그룹 내 구성원끼리는 해당 문의 내역을 함께 확인할 수 있습니다.

담당자: 실제 업무 처리를 담당하실 분의 성함을 입력해 주세요. (SR 등록자와 실제 조치를 받을 실무자가 다를 경우를 위해 필요합니다.)

연락처: 신속하고 원활한 기술 지원 커뮤니케이션을 위해 위 담당자의 연락처를 기재해 주세요.

문의 고객사: 기술 지원을 제공받는 고객사(회사명)의 이름을 정확히 입력해 주세요.

Accordion 버전: 현재 사용 중인 Accordion의 버전을 선택해 주세요. (정확한 버전을 확인하기 어려울 경우 혹은 버전이 없는 경우 기타로 선택해 주시면 됩니다.)

문의 영역: 문의하시는 내용에 해당하는 카테고리를 선택해 주세요.

문의 내용: 문의 상세 내용, 발생한 문제 증상, 기대하는 결과 등을 최대한 구체적으로 작성해 주세요.

첨부 파일 : 문의에 대해서 참고할 수 있는 이미지나 파일을 업로드해 주새요.
