## 이벤트 로그

이벤트 로그는 클러스터에서 발생한 Kubernetes 이벤트를 조회하는 메뉴입니다.
파드 스케줄링, 이미지 풀링, 컨테이너 생성 실패 같은 리소스 수명 주기 이벤트를 확인할 수 있습니다.

![eventlog_interface.png](/assets/images/v3/monitoring/eventlog_interface.png){: width="1200" }

---

## 필터 조건

| 항목 | 설명 |
|---|---|
| 네임스페이스 | 이벤트가 발생한 네임스페이스 |
| 종류 | 이벤트 대상 리소스 종류 |
| 이름 | 이벤트 대상 리소스 이름 |
| 타입 | Normal, Warning 등 이벤트 타입 |
| 원인 | 이벤트 발생 원인 |
| 메시지 | 이벤트 메시지 키워드 |

![eventlog_kind.png](/assets/images/v3/monitoring/eventlog_kind.png)

---

## 화면 구성

| 영역 | 설명 |
|---|---|
| 검색 결과 | 현재 조건에 해당하는 이벤트 로그 건수를 표시합니다. |
| 로그 카운트 차트 | 시간대별 이벤트 발생 건수를 표시합니다. |
| 이벤트 로그 테이블 | 개별 이벤트 로그를 목록으로 표시합니다. |
| 상세 정보 | 선택한 이벤트 원문 정보를 표시합니다. |

---

## 테이블 항목

| 항목 | 설명 |
|---|---|
| 번호 | 로그 순번 |
| 시간 | 이벤트 발생 일시 |
| 네임스페이스 | 이벤트가 속한 네임스페이스 |
| 이름 | 이벤트 대상 이름 |
| 카운트 | 동일 이벤트 누적 발생 횟수 |
| 종류 | 이벤트 대상 리소스 종류 |
| 타입 | 이벤트 타입 |
| 원인 | 이벤트 발생 원인 |
| 메시지 | 이벤트 상세 메시지 |

---

## 상세 정보

테이블에서 로그를 선택하면 이벤트 오브젝트의 상세 정보가 표시됩니다.
이벤트 대상 리소스, 발생 원인, 메시지, 최초/마지막 발생 시각, 누적 횟수를 확인할 수 있습니다.

![eventlog_more.png](/assets/images/v3/monitoring/eventlog_more.png){: width="1200" }
