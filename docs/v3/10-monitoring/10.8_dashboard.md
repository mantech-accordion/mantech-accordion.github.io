## 대시보드

대시보드는 모니터링 데이터를 시각화하기 위한 화면입니다.
클러스터와 서비스 상태를 필요한 지표 중심으로 확인할 수 있습니다.

![dashboard_interface.png](/assets/images/v3/monitoring/dashboard_interface.png){: width="1200" }


---

## 화면 구성

| 항목 | 설명 |
|---|---|
| 대시보드 목록 | 생성되었거나 제공되는 대시보드 목록을 확인합니다. |
| 대시보드 상세 | 선택한 대시보드의 그래프와 지표를 확인합니다. |
| 조회 조건 | 대시보드의 조회 기간과 갱신 조건을 설정합니다. |
