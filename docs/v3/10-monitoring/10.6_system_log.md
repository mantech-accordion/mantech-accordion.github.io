## 시스템 로그

시스템 로그는 클러스터의 시스템 컴포넌트 컨테이너 로그를 조회하는 메뉴입니다.
시스템 동작 상태, 오류 메시지, 디버그 정보를 확인할 수 있습니다.

기본적으로 `kube-system`, `acc-system` 네임스페이스의 컨테이너 로그를 제공합니다.

![systemlog_interface.png](/assets/images/v3/monitoring/systemlog_interface.png){: width="1200" }

---

## 필터 조건

![systemlog_kind.png](/assets/images/v3/monitoring/systemlog_kind.png)

| 항목 | 설명 |
|---|---|
| 네임스페이스 | 로그가 발생한 네임스페이스 |
| 노드명 | 로그가 발생한 노드 |
| 파드명 | 로그가 발생한 파드 |
| 컨테이너명 | 로그가 발생한 컨테이너 |
| 로그 | 로그 메시지 키워드 |



---

## 화면 구성

| 영역 | 설명 |
|---|---|
| 검색 결과 | 현재 조건에 해당하는 시스템 로그 건수를 표시합니다. |
| 로그 카운트 차트 | 시간대별 시스템 로그 발생 건수를 표시합니다. |
| 시스템 로그 테이블 | 개별 로그를 목록으로 표시합니다. |
| 상세 정보 | 선택한 컨테이너의 로그 상세 내용을 표시합니다. |

---

## 테이블 항목

| 항목 | 설명 |
|---|---|
| 번호 | 로그 순번 |
| 시간 | 로그 발생 일시 |
| 노드 | 로그가 발생한 노드 |
| 네임스페이스 | 로그가 발생한 네임스페이스 |
| 파드 | 로그가 발생한 파드 |
| 컨테이너 | 로그가 발생한 컨테이너 |
| 로그 | 로그 메시지 본문 |

---

## 상세 정보

테이블에서 로그를 선택하면 선택한 컨테이너의 로그 상세 정보가 표시됩니다.
조회 시간 범위 내 로그가 시간순으로 표시되어 오류 전후의 흐름을 확인할 수 있습니다.

![systemlog_more.png](/assets/images/v3/monitoring/systemlog_more.png){: width="1200" }
