## 알림 로그

알림 로그는 클러스터에서 발생한 임계치 알림, 이상탐지 알림, 장애예측 알림을 조회하는 메뉴입니다.
알림 수준, 발생 리소스, 수신자, 발송 결과를 확인할 수 있습니다.

![alert-log_interface.png](/assets/images/v3/monitoring/alert-log_interface.png){: width="1200" }

---

## 필터 조건

![alert-log_kind.png](/assets/images/v3/monitoring/alert-log_kind.png)

| 항목 | 설명 |
|---|---|
| 알림수준 | Info, Warning, Critical 등 알림 수준 |
| 네임스페이스 | 알림이 발생한 네임스페이스 |
| 종류 | 알림이 발생한 리소스 종류 |
| 이름 | 알림 정책 또는 리소스 이름 |
| 알림정보 | 알림 상세 정보 키워드 |
| 수신자 | 알림 수신 대상 |
| 결과 | 알림 발송 결과 |

---

## 화면 구성

| 영역 | 설명 |
|---|---|
| 검색 결과 | 현재 조건에 해당하는 알림 로그 건수를 표시합니다. |
| 로그 카운트 차트 | 시간대별 알림 발생 건수를 표시합니다. |
| 알림 로그 테이블 | 개별 알림 로그를 목록으로 표시합니다. |
| 상세 정보 | 선택한 알림의 상세 내용을 표시합니다. |

---

## 테이블 항목

| 항목 | 설명 |
|---|---|
| 번호 | 로그 순번 |
| 시간 | 알림 발생 일시 |
| 알림수준 | 알림 경보 수준 |
| 네임스페이스 | 알림 발생 네임스페이스 |
| 종류 | 알림이 발생한 리소스 종류 |
| 이름 | 알림 정책 또는 리소스 이름 |
| 알림정보 | 알림 발생 상세 정보 |
| 수신자 | 알림 수신 대상 |
| 결과 | 알림 발송 결과 |

---

## 상세 정보

테이블에서 로그를 선택하면 상세 정보가 표시됩니다.
일반 알림은 JSON 형식의 알림 오브젝트를 보여주고, 이상탐지·장애예측 알림은 관련 메트릭 그래프를 함께 보여줍니다.
