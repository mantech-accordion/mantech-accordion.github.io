## 시스템

시스템은 클러스터 리소스의 상태와 사용량을 확인하는 메뉴입니다.
개요, 노드, 네임스페이스, 파드, 워크로드, GPU 단위로 모니터링 정보를 제공합니다.

![system-overview.png](/assets/images/v3/monitoring/system_overview.png){: width="1200" }

---

## 공통 제어 영역

시계열 그래프가 있는 화면에서는 조회 기간, 새로고침, 자동 갱신 주기를 설정할 수 있습니다.

| 항목 | 설명 |
|---|---|
| 조회 기간 | 최근 5분, 최근 1시간 등 조회 범위를 선택합니다. |
| 새로고침 | 현재 조건으로 데이터를 즉시 다시 조회합니다. |
| 자동 갱신 주기 | 끄기, 5초, 10초, 30초, 1분 등 자동 갱신 주기를 선택합니다. |
| 조회 범위(Step) | 그래프 데이터 간격을 설정합니다. 자동 설정도 가능합니다. |

## 개요

개요 탭은 클러스터 전체 상태를 빠르게 확인하는 화면입니다.
무대형 개요와 상세 개요를 전환하여 볼 수 있습니다.

| 구분 | 설명 |
|---|---|
| 무대형 개요 | 노드와 파드 배치 상태를 시각적으로 확인합니다. |
| 상세 개요 | 보기 기준, 세부 기준, 메트릭을 선택해 리소스 상태를 카드 형태로 확인합니다. |

## 노드

노드 탭은 노드별 CPU, 메모리, 디스크, 네트워크 사용량을 제공합니다.
특정 노드에 부하가 집중되는지 확인할 때 사용합니다.

![system-node.png](/assets/images/v3/monitoring/system_node.png){: width="1200" }

| 항목 | 설명 |
|---|---|
| CPU 사용률 | 노드별 평균 CPU 사용률 |
| 메모리 사용률 | 노드별 평균 메모리 사용률 |
| 디스크 사용률 | 노드별 평균 디스크 사용률 |
| 디스크 읽기/쓰기 | 노드별 디스크 I/O |
| 네트워크 송신/수신 | 노드별 네트워크 사용량 |

---

## 네임스페이스

네임스페이스 탭은 네임스페이스별 리소스 사용량을 제공합니다.
어떤 네임스페이스에서 리소스를 많이 사용하는지 비교할 때 사용합니다.

![system-namespace.png](/assets/images/v3/monitoring/system_namespace.png){: width="1200" }

| 항목 | 설명 |
|---|---|
| CPU 사용량 | 네임스페이스별 CPU 사용량 |
| 메모리 사용량 | 네임스페이스별 메모리 사용량 |
| 디스크 읽기/쓰기 | 네임스페이스별 디스크 I/O |
| 네트워크 송신/수신 | 네임스페이스별 네트워크 사용량 |

---

## 파드

파드 탭은 파드 단위의 리소스 사용량과 상태를 제공합니다.
재시작 횟수와 Ready 상태를 함께 확인할 수 있어 장애 파드를 찾을 때 유용합니다.

![system-pod.png](/assets/images/v3/monitoring/system_pod.png){: width="1200" }

| 항목 | 설명 |
|---|---|
| CPU/메모리 사용량 | 파드별 평균 리소스 사용량 |
| 디스크 읽기/쓰기 | 파드별 디스크 I/O |
| 네트워크 송신/수신 | 파드별 네트워크 사용량 |
| 재시작 | 컨테이너 재시작 횟수 |
| 준비 | Ready 컨테이너 수 / 전체 컨테이너 수 |

---

## 워크로드

레플리카셋, 디플로이먼트, 스테이트풀셋, 데몬셋, 잡, 크론잡 탭은 각 워크로드 단위로 리소스 사용량과 준비 상태를 제공합니다.

![system-workload.png](/assets/images/v3/monitoring/system_workload.png){: width="1200" }

| 탭 | 설명 |
|---|---|
| 레플리카셋 | ReplicaSet 단위 리소스 상태 확인 |
| 디플로이먼트 | Deployment 단위 리소스와 Ready 상태 확인 |
| 스테이트풀셋 | StatefulSet 단위 리소스와 Ready 상태 확인 |
| 데몬셋 | DaemonSet 단위 리소스와 Ready 상태 확인 |
| 잡/크론잡 | Job, CronJob 실행 리소스 상태 확인 |

---

## GPU

GPU 탭은 GPU 자원 상태를 확인하는 화면입니다.
GPU 사용률, 메모리 사용률, 온도, 전력, XID 오류, 스로틀링 정보를 확인할 수 있습니다.

![system-gpu.png](/assets/images/v3/monitoring/system_gpu.png){: width="1200" }

| 메뉴 | 설명 |
|---|---|
| 전체 개요 | 클러스터 전체 GPU 상태 요약 |
| 개별 GPU 현황 | GPU 장치 단위 상세 현황 |
| 노드 기준 분석 | 노드별 GPU 사용량 비교 |
| 네임스페이스 기준 분석 | 네임스페이스별 GPU 사용량 비교 |
| 워크로드 기준 분석 | 워크로드별 GPU 사용량 비교 |
| 파드 기준 분석 | 파드별 GPU 사용량 확인 |
