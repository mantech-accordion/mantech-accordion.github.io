### 헬름 리포지터리 관리

아코디언에서는 헬름을 사용하기 위해서는 먼저 헬름 리포지터리를 등록해야합니다. 기존 사용 중이었던 헬름 리포지터리도 간편하게 등록 및 관리 가능합니다.

![helmrepo_list.png](/assets/images/v3/application/helm/helmrepo_list.png){: width="1200" }

---

### 헬름 리포지터리 등록

클러스터 설정 탭의 헬름 리포지터리 메뉴에 접근하여 좌측 상단의 `+ 생성` 버튼을 선택하면 헬름 리포지터리 정보를 입력하는 페이지로 이동합니다.

![helmrepo_create.png](/assets/images/v3/application/helm/helmrepo_create.png){: width="1200" }

| 항목        |  설명  |
|:------------|:-------|
| 이름        | 헬름 리포지터리 이름 (필수) | 
| URL         | 헬름 리포지터리 URL (필수) | 
| 사용자명    | 헬름 리포지터리 아이디 (옵션)  | 
| 비밀번호    | 헬름 리포지터리 패스워드 (옵션)  | 

---

### 헬름 리포지터리 업데이트

생성 버튼 아래의 업데이트 버튼을 선택하면 헬름 리포지터리의 차트 정보를 최신으로 업데이트할 수 있습니다.

![helmrepo_update.png](/assets/images/v3/application/helm/helmrepo_update.png)

---

### 헬름 리포지터리 삭제

삭제하려는 리포지터리를 선택하고 우측 상단의 `삭제` 버튼을 선택합니다.
- 기본 제공하는 accordion 리포지터리는 삭제할 수 없습니다.

![helmrepo_delete.png](/assets/images/v3/application/helm/helmrepo_delete.png)

---

### 아코디언 리포지터리

아코디언에서 제공하고 있는 accordion 비공개 저장소는 Golang으로 작성된 오픈 소스 헬름 리포지터리 서버입니다.

Google Cloud Storage, Amazon S3, Azure Blob Storage, Openstack Object Storage 를 포함한 클라우드 스토리지 백엔드를 지원합니다.

![acc_repo.png](/assets/images/v3/application/helm/acc_repo.png){: width="1200" }
