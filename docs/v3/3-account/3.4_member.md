# **멤버**
{: .no_toc }

## Table of contents
{: .no_toc .text-delta }

1. TOC
{:toc}

---

멤버는 사용자(또는 그룹)과 권한을 바인딩하는 역할을 합니다.<br>
이러한 멤버는 스코프 별로 따로 관리되며 각각 글로벌/클러스터/네임스페이스 멤버라고 합니다.<br>
각 스코프 권한에서는 해당 스코프 이하의 메뉴에 대해 바인딩할 수 있습니다. 예를 들면 아래와 같습니다.

```
- 글로벌 멤버
    - 글로벌 메뉴에 대한 롤 할당 가능 (글로벌 스코프)
    - 클러스터 메뉴에 대한 롤 할당 가능 (클러스터 스코프)
    - 네임스페이스 메뉴에 대한 롤 할당 가능 (네임스페이스 스코프)
- 클러스터 멤버
    - 클러스터 메뉴에 대한 롤 할당 가능 (클러스터 스코프)
    - 네임스페이스 메뉴에 대한 롤 할당 가능 (네임스페이스 스코프)
- 네임스페이스 멤버
    - 네임스페이스 메뉴에 대한 롤 할당 가능 (네임스페이스 스코프)
```
