# **그룹**
{: .no_toc }

그룹은 사용자 집합을 의미합니다. 그룹 메뉴에서 그룹을 생성, 그룹 정보를 수정, 그룹을 삭제할 수 있습니다.<br>
관리자는 `멤버` 메뉴에서 사용자 또는 그룹에 권한을 부여하게 됩니다.<br>
아코디언에서는 그룹 단위로 사용자를 묶어 관리하며, 그룹에 부여된 권한을 구성원들이 상속받게 됩니다.

## 그룹 목록

좌측의 그룹 목록에서는 그룹명, Display Name, 요약 정보가 리스트에 간략히 노출되고, 검색도 가능합니다. 또한 상단의 `+ 생성` 버튼을 클릭하여 신규 그룹을 등록할 수 있습니다. 필수 항목인 이름 및 기타 선택항목을 작성 후 `그룹 생성` 버튼을 선택하면 됩니다.

![group-list.png](/assets/images/v3/auth/group-list.png){: width="1200" }

**그룹 상세 정보**

우측의 그룹 상세 정보는 `개요`와 `권한` 탭으로 구성되어 있습니다.

### 개요

**- 개요**

그룹 등록 정보를 나타내며, `그룹 수정` 버튼을 누를 시 그룹 정보 수정이 가능합니다.

| 항목  | 설명 |
|---|---|
| 그룹명   | 그룹 이름  |
| UID  | 그룹의 고유 아이디  |
| 요약  | 그룹의 요약된 정보  |
| Display Name  | 사용자, 글로벌 멤버, 클러스터 멤버, 네임스페이스 멤버에서 해당 그룹이 표시되는 이름  |
| 설명  | 그룹의 상세 설명  |

**- 구성원**

그룹 내 속한 구성원 정보를 한 눈에 볼 수 있습니다. 사용자 목록과 각 사용자의 계정 정보도 확인 가능합니다.

### 권한

| 항목  | 설명 |
|---|---|
| 내보내기   | 권한을 csv 파일로 내보낸다.   |
| 글로벌 권한<br> 클러스터 권한<br> 네임스페이스 권한<br>   | `글로벌 권한`, `클러스터 권한`, `네임스페이스 권한` 세 개의 탭으로 나누어서 볼 수 있고, 각 탭 클릭 시 해당 스코프의 권한 목록이 노출된다. 또한 각 스코프마다 현재 그룹이 몇 개의 권한을 가지고 있는지 확인할 수 있다.<br> 현재 그룹이 해당 권한을 가지고 있다면 `v` 로 체크된다.<br> 권한 목록의 권한 이름을 선택할 경우 해당 권한 페이지로 이동된다. |

## 그룹 생성

그룹을 생성하기 위해서는 좌측 상단의 `+ 생성` 버튼을 선택합니다.

![group-create.png](/assets/images/v3/auth/group-create.png){: width="1200" }

## 그룹에 사용자 추가

그룹은 사용자 집합으로 사용자 목록을 관리합니다. 사용자 메뉴에서 그룹에 추가할 특정 사용자를 목록에서 클릭합니다. 우측의 `계정 정보` - `그룹` 항목에서 `그룹 수정` 버튼을 누른 뒤에 사용자에게 추가하려는 그룹을 선택하여 그룹에 사용자를 추가합니다. 한 사용자는 동시에 여러 그룹에 속할 수 있습니다.

![group-adduser.png](/assets/images/v3/auth/group-adduser.png){: width="1200" }

사용자에 그룹 추가 후 해당 그룹을 확인하면 그룹의 구성원에 사용자가 추가된 것을 확인할 수 있습니다.

![group-adduser2.png](/assets/images/v3/auth/group-adduser2.png){: width="1200" }

## 그룹에 글로벌 권한 부여

권한 부여는 멤버를 통해서만 권한을 부여할 수 있습니다. 권한과 멤버는 각각 글로벌/클러스터/네임스페이스가 존재합니다.

글로벌 권한을 부여하려면 글로벌 권한에서 부여하려는 권한을 생성합니다.

[글로벌 권한 생성] 참고

글로벌 권한을 생성한 후 글로벌 멤버의 좌측 목록 탭의 `+ 생성` 버튼을 클릭합니다. 그리고 `할당 대상 선택` 항목에서 그룹을 클릭하고, 권한을 부여하려는 그룹을 선택 후 그룹에 부여하려는 권한을 아래에서 선택하여 글로벌 멤버를 생성합니다. 클러스터/네임스페이스도 글로벌 권한 부여와 동일한 방식으로 권한 부여합니다.

![global-member-group.png](/assets/images/v3/auth/global-member-group.png){: width="1200" }

## 그룹 삭제

그룹을 삭제하기 위해서는 목록에서 삭제할 대상이 되는 그룹을 선택하고 우측 상단의 `삭제` 버튼을 선택합니다.<br> 
이후 나타나는 모달에서 그룹 이름을 입력하고 `삭제` 버튼을 선택하면 완료됩니다.<br>
해당 그룹이 글로벌 멤버, 클러스터 멤버, 네임스페이스 멤버에 설정되어 있을 경우, 해당 멤버에서 제외됩니다.<br>

![group-delete.png.png](/assets/images/v3/auth/group-delete.png){: width="400" }
