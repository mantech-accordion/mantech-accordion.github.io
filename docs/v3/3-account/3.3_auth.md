# **권한**
{: .no_toc }

## Table of contents
{: .no_toc .text-delta }

1. TOC
{:toc}

---

권한은 여러 메뉴와 롤의 집합을 의미합니다.<br>
이러한 권한은 스코프 별로 따로 관리되며 각각 글로벌/클러스터/네임스페이스 권한이라고 합니다.<br>
각 스코프 권한에서는 해당 스코프 이하의 메뉴에 대해 바인딩할 수 있습니다. 예를 들면 아래와 같습니다.

```
- 글로벌 권한
    - 글로벌 메뉴에 대한 롤 설정 가능 (글로벌 스코프)
    - 클러스터 메뉴에 대한 롤 설정 가능 (클러스터 스코프)
    - 네임스페이스 메뉴에 대한 롤 설정 가능 (네임스페이스 스코프)
- 클러스터 권한
    - 클러스터 메뉴에 대한 롤 설정 가능 (클러스터 스코프)
    - 네임스페이스 메뉴에 대한 롤 설정 가능 (네임스페이스 스코프)
- 네임스페이스 권한
    - 네임스페이스 메뉴에 대한 롤 설정 가능 (네임스페이스 스코프)
```
