# **글로벌 권한**
{: .no_toc }

글로벌 권한은 모든 메뉴에 대한 권한을 설정할 수 있습니다.

## Table of contents
{: .no_toc .text-delta }

1. TOC
{:toc}

---

### 메뉴이동
`글로벌 모드 on` ➡ `글로벌 설정` ➡ `글로벌 권한`

![global.png](/assets/images/v3/auth/global.png)

---

### 글로벌 권한 목록

메뉴에서 다음과 같이 글로벌 권한 목록과 권한 정보들을 조회할 수 있습니다. 하나의 권한은 하나 이상의 메뉴와 롤로 구성됩니다. 

![auth-list.png](/assets/images/v3/auth/auth-list.png){: width="1200" }

---

### 글로벌 권한 등록

글로벌 권한을 등록하기 위해서는 좌측 상단의 `+ 생성` 버튼을 선택하고 권한의 이름과 권한에 포함할 메뉴 및 롤을 추가하면 됩니다.
- 권한은 admin/viewer 중 선택할 수 있습니다.

![auth-create.png](/assets/images/v3/auth/auth-create.png){: width="1200" }

---

### 글로벌 권한 수정

글로벌 권한을 수정하기 위해서는 수정하려는 권한을 선택하고 우측 상단의 `수정` 버튼을 선택하여 수정 화면으로 이동합니다. 그리고 메뉴을 추가하거나 삭제, 또는 메뉴에 대한 롤을 변경합니다. 마지막으로 우측 상단의 `글로벌 권한 수정` 버튼을 선택하여 변경 사항을 권한에 반영합니다.

![auth-update.png](/assets/images/v3/auth/auth-update.png){: width="1200" }

---

### 글로벌 권한 삭제

글로벌 권한을 삭제하기 위해서는 삭제하려는 권한을 선택하고 우측 상단의 삭제 버튼을 선택합니다.
이후 표시되는 모달에서 권한 이름을 입력하고 삭제하기 버튼을 선택합니다.

![auth-delete.png](/assets/images/v3/auth/auth-delete.png){: width="400" }
