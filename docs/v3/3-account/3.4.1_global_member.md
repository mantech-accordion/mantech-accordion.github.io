## 글로벌 멤버
글로벌 멤버는 사용자와 그룹의 글로벌 권한을 관리합니다.

---

### 메뉴 이동
`글로벌 모드 on` ➡ `글로벌 설정` ➡ `글로벌 멤버`

![global-member.png](/assets/images/v3/auth/global-member.png)

---

### 글로벌 멤버

![member-list.png](/assets/images/v3/auth/member-list.png){: width="1200" }

| 종류  | 설명 |
|---|---|
| 사용자  | 개별 사용자의 글로벌 권한 관리 |
| 그룹 | 그룹 별 글로벌 권한 관리 <br> (그룹에 있는 사용자들에게 모두 권한을 부여하는 것과 같은 효과)  |

---

### 글로벌 멤버 생성

사용자(또는 그룹) 목록 좌측 상단의 `+ 생성` 버튼을 선택하여 생성합니다.

![member-add.png](/assets/images/v3/auth/member-add.png){: width="1200" }

---

### 글로벌 멤버 수정

사용자(또는 그룹) 우측 상단의 `수정` 버튼으로 해당 사용자(또는 그룹)의 권한을 수정할 수 있습니다.

![member-update.png](/assets/images/v3/auth/member-update.png){: width="1200" }

---

### 글로벌 멤버 삭제

사용자(또는 그룹)의 우측 상단 삭제 버튼을 선택하면 멤버를 삭제할 수 있습니다.

![member-delete.png](/assets/images/v3/auth/member-delete.png){: width="400" }
