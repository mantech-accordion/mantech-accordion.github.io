## 네임스페이스 멤버
네임스페이스 사용자와 그룹의 권한을 관리합니다.

---

## 메뉴이동
`글로벌 모드 off` ➡ `네임스페이스` 선택 ➡ `네임스페이스 설정` ➡ `네임스페이스 멤버`

![ns-member](/assets/images/v3/auth/ns-member.png)

---

### 네임스페이스 멤버

![ns-member-list.png](/assets/images/v3/auth/ns-member-list.png){: width="1200" }

| 종류  | 설명 |
|---|---|
| 사용자  | 개별 사용자의 네임스페이스 권한 관리 |
| 그룹 | 그룹 별 네임스페이스 권한 관리 <br> (그룹에 있는 사용자들에게 모두 권한을 부여하는 것과 같은 효과)  |

---

### 네임스페이스 멤버 생성

사용자(또는 그룹) 목록 좌측 상단의 `+ 생성` 버튼을 선택하여 생성합니다.

![ns-member-add.png](/assets/images/v3/auth/ns-member-add.png){: width="1200" }

---

### 네임스페이스 멤버 수정

사용자(또는 그룹) 우측 상단의 `수정` 버튼으로 해당 사용자(또는 그룹)의 권한을 수정할 수 있습니다.

![ns-member-update.png](/assets/images/v3/auth/ns-member-update.png){: width="1200" }

---

### 네임스페이스 멤버 삭제

사용자(또는 그룹)의 우측 상단 삭제 버튼을 선택하면 멤버를 삭제할 수 있습니다.

![ns-member-delete.png](/assets/images/v3/auth/ns-member-delete.png){: width="400" }
