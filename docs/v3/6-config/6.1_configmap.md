## 컨피그맵(Configmap)

컨피그맵은 키-값 쌍으로 민감한 정보가 아닌 데이터를 저장하는 데 사용하는 API 오브젝트입니다. 파드는 볼륨에서 환경 변수, 커맨드-라인 인수 또는 구성 파일로 컨피그맵을 사용할 수 있습니다.

컨피그맵을 사용하면 컨테이너 이미지에서 환경별 구성을 분리하여, 애플리케이션을 쉽게 이식할 수 있습니다.

> **주의:**
>
> 컨피그맵은 보안 또는 암호화를 제공하지 않습니다. 저장하려는 데이터가 기밀인 경우, 컨피그맵 대신 [시크릿(Secret)](../6-config/6.2_secret.md) 또는 추가(써드파티) 도구를 사용하여 데이터를 비공개로 유지할 수 있습니다.
{: .callout-warning }

## 컨피그맵 생성

`+생성` 을 선택하면 나타나는 모달에서 쿠버네티스 컨피그맵 리소스 정보를 입력하여 생성할 수 있습니다. 생성 시에는 FORM/YAML로 입력할 수 있습니다.

![config-003.png](/assets/images/v3/config/configmap/cm_create.png)

---

## 컨피그맵 수정

수정하려는 컨피그맵을 선택하고 우측 상단의 수정 버튼을 클릭합니다.

설정 탭에서 데이터를 수정할 수 있습니다.

![config-004.png](/assets/images/v3/config/configmap/cm_edit.png)

---

## 컨피그맵 삭제

삭제하려는 컨피그맵을 선택하고 우측의 삭제 버튼을 선택합니다. 모달에서 컨피그맵 이름을 입력하여 삭제합니다.

![config-005.png](/assets/images/v3/config/configmap/cm_delete.png)
