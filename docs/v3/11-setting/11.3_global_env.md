## 글로벌 환경설정

글로벌 환경 설정에서는 계정 세션, 비밀번호 정책 등 계정 관련 옵션을 설정할 수 있습니다.

![global-env-setting.png](/assets/images/v3/setting/global-env-setting.png)

**세션 관리**<br>

| 분류 | 설명 |
| :--- | :--- |
| 유휴 세션 타임아웃 (초) | 마지막 요청 이후 아무 동작이 없으면 세션이 만료되는 시간(초). |
| 최대 세션 타임아웃 (초) | 활동 여부와 무관하게 세션이 유지되는 최대 시간(초). |

**브라우저**<br>

| 분류 | 설명 |
| :--- | :--- |
| 사용자당 최대 세션 수 | 한 사용자가 동시에 유지할 수 있는 최대 세션 수. 초과 시 새 세션이 연결될 수 없습니다. |

**패스워드 정책**<br>

| 분류 | 설명 |
| :--- | :--- |
| 최소 길이 | 비밀번호의 최소 문자 수. |
| 패스워드 만료 (일) | 비밀번호 변경 주기(일). 빈 값 입력 시 만료 정책을 적용하지 않습니다. |
| 대문자 필수 개수 | 비밀번호에 포함해야 하는 대문자(A-Z) 최소 개수. |
| 소문자 필수 개수 | 비밀번호에 포함해야 하는 소문자(a-z) 최소 개수. |
| 숫자 필수 개수 | 비밀번호에 포함해야 하는 숫자(0-9) 최소 개수. |
| 특수 문자 필수 개수 | 비밀번호에 포함해야 하는 특수 문자 최소 개수. |
| 이전 패스워드 재사용 방지 개수 | 재사용을 방지할 이전 비밀번호 개수. 빈 값 입력 시 비활성화됩니다.<br>(0 입력 시에도 직전 1개 비밀번호는 재사용이 방지됩니다.) |
| 패스워드 입력 패턴 | 비밀번호가 충족해야 하는 정규표현식 패턴. (예: `^[a-zA-Z0-9]+$`) |
