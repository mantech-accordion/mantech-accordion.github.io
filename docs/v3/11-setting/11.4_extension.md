## 확장 메뉴 (Extensions)
확장 메뉴는 아코디언 클러스터 내부에 배포된 다양한 서비스의 웹 UI를 외부 브라우저를 통해 간편하게 모니터링하고 조회할 수 있도록 연결해 주는 기능입니다.

사전 필수 작업: 본 기능을 사용하려면 먼저 [클러스터 환경설정 > 확장 기능] 메뉴에서 대상 서비스를 등록해야 합니다.


---
## 메뉴이동
`클러스터 설정` ➡ `클러스터 환경설정` 

`확장` ➡ `등록한 서비스 탭`

![cluster-setting.png](/assets/images/v3/extension/cluster-setting.png)

---

## 확장 기능 생성
클러스터 설정에서 '확장 기능 생성' 버튼을 이용하여 등록합니다.
외부 서비스의 URL을 등록하여 아코디언 사이드바 메뉴에 바로가기를 추가할 수 있습니다.

![extension-create.png](/assets/images/v3/extension/extension-create.png){: width="1200" }

생성하면 권한 설정 화면으로 이동합니다.
클러스터 UI 템플릿 권한 및 네임스페이스/클러스터 권한을 추가하여 메뉴바에 확장 기능이 생성되었는지 확인합니다.

![extension-access.png](/assets/images/v3/extension/extension-access.png){: width="1200" }

![extension-access-1.png](/assets/images/v3/extension/extension-access-1.png){: width="1200" }

---

## 확장 기능 조회
권한을 부여하고나면 사용자가 좌측 확장 메뉴에서 항목을 할 수 있고, 우측 화면에 해당 서비스의 UI가 표시됩니다.

![extension.png](/assets/images/v3/extension/extension.png){: width="1200" }


---

## 확장 기능 수정
확장 기능 목록에서 수정(연필 아이콘) 버튼을 클릭하면 수정 화면으로 이동한다. 수정 화면은 생성 화면과 동일한 구조이며, 기존에 입력된 정보가 표시된다.

![extension-modi.png](/assets/images/v3/extension/extension-modi.png){: width="1200" }

---

## 확장 기능 삭제
확장 기능 목록에서 삭제(휴지통 아이콘) 버튼을 클릭하면 해당 확장 기능이 삭제된다.

![extension-del.png](/assets/images/v3/extension/extension-del.png){: width="1200" }

## 제한 사항
확장 메뉴는 내부 <font color="steelblue">관리 조회 목적</font>으로 <font color="indianred">일반 웹 서비스 제공</font> 용도로 사용하는 것은 <font color="indianred">권장하지 않습니다</font>.
본 기능은 리버스 프록시를 경유하는 방식으로 동작하며, 상대 경로(Relative Path) 기반으로 구현된 애플리케이션에서만 정상 동작을 보장합니다.

다음과 같은 경우 동작을 보장하지 않는다.

- 절대 경로(Absolute Path) 사용
- 복잡한 SPA(React, Vue 등) 구조
- WebSocket 의존 기능
- 외부 Redirect가 필요한 구조
