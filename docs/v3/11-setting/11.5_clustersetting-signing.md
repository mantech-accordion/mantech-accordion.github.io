## 이미지 서명 보안 설정
이미지 서명 보안 설정 토글을 활성화하면 체크 모드와 적용 범위 설정이 표시됩니다.
설정을 변경한 후 저장 버튼을 클릭하여 저장합니다.


![imagesign.png](/assets/images/v3/setting/imagesign.png){: width="1200" }


### 체크 모드

이미지 서명 검증 방식을 선택합니다.
체크 모드 변경 시, 재기동되는 동안 일시적으로 레지스트리 기능이 정상적으로 동작하지 않을 수 있습니다. 
Enforcing 모드의 경우 서명되지 않은 이미지 배포는 모두 차단되므로 주의해서 설정해야 합니다.


| 모드        | 항목    | 설명              |
|:----------|----------|--------------------|
| Enforcing | 실행 거부 | Enforcing 모드에서는 서명되지 않은 이미지의 배포가 차단됩니다. 프로덕션 환경에 권장합니다. |
| Permissive | 로그 기록 | Permissive 모드에서는 서명되지 않은 이미지도 실행되지만, 모든 위반 사항이 이벤트 로그에 기록됩니다. |

### 적용 모드
이미지 서명 보안 정책이 적용될 범위를 선택합니다. 체크박스로 클러스터 또는 네임스페이스를 선택할 수 있습니다.

| 모드     | 설명                                                                                                |
|:-------|---------------------------------------------------------------------------------------------------|
| 클러스터   | 클러스터 전체에 이미지 서명 보안 정책을 적용합니다. 선택 시 적용할 레지스트리를 선택할 수 있습니다 (복수 선택 가능).                              |
| 네임스페이스 | 특정 네임스페이스에 이미지 서명 보안 정책을 적용합니다. 선택 시 네임스페이스 드롭다운이 표시되며, 적용할 네임스페이스와 레지스트리를 선택할 수 있습니다 (복수 선택 가능). |
